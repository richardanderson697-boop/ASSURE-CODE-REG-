## üèÉ Path A: Dashboard Only (15 Minutes)

### Step 1: Clone & Setup (3 min)
```bash
# Create project directory
mkdir compliance-dashboard
cd compliance-dashboard

# Create files (copy from artifacts):
# - main.py
# - database.py
# - simplified_integration.py
# - requirements.txt
# - setup_database.py

# Install dependencies
pip install -r requirements.txt
python -m playwright install chromium
```

### Step 2: Start Database (2 min)
```bash
# Using Docker (easiest)
docker run -d \
  --name compliance-db \
  -e POSTGRES_PASSWORD=secure123 \
  -e POSTGRES_USER=scraper \
  -e POSTGRES_DB=compliance_dashboard \
  -p 5432:5432 \
  postgres:15-alpine

# OR use Railway/Supabase/Neon for managed PostgreSQL
```

### Step 3: Configure Environment (2 min)
```bash
# Create .env file
cat > .env << EOF
DATABASE_URL=postgresql+asyncpg://scraper:secure123@localhost:5432/compliance_dashboard
OPENAI_API_KEY=sk-your-key-here
ENVIRONMENT=production
ASSURE_CODE_ENABLED=false
EOF
```

### Step 4: Initialize Database (2 min)
```bash
python setup_database.py init

# üìã SAVE THE API KEY SHOWN!
# It looks like: rcp_abc123xyz...
```

### Step 5: Start Application (1 min)
```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```

### Step 6: Test It Works (5 min)
```bash
# Test scraping
curl -X POST http://localhost:8000/api/v1/scrape \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.sec.gov/files/rules/final/2023/34-98908.pdf",
    "jurisdiction": "US",
    "category": "finance"
  }'

# Check status
curl http://localhost:8000/health
```

### Step 7: Access Dashboard
Open browser to: `http://localhost:8000`

**‚úÖ You're Done! Dashboard is running.**

## üîÑ Path B: Complete Integration (45 Minutes)

### Prerequisites
- Dashboard deployed (Path A)
- ASSURE-CODE platform running
- Both have public URLs

### Step 1: Enable Integration on Scraper (10 min)
```bash
# Update .env
ASSURE_CODE_ENABLED=true
ASSURE_CODE_API_URL=https://your-assure-code.replit.app
ASSURE_CODE_API_KEY=generate-secure-key-here

# Restart application
```

### Step 2: Add Webhook to ASSURE-CODE (15 min)
```bash
# 1. Add complianceScraperWebhook.ts to your ASSURE-CODE
# 2. Register route in server/index.ts
# 3. Add to .env:
COMPLIANCE_SCRAPER_WEBHOOK_URL=https://your-scraper.com/webhook
COMPLIANCE_SCRAPER_API_KEY=same-key-as-above
```

### Step 3: Test Integration (10 min)
```bash
# 1. Create scrape job on dashboard
# 2. Wait for completion
# 3. Send to ASSURE-CODE:
curl -X POST http://scraper/api/v1/assure-code/send/{JOB_ID} \
  -H "Authorization: Bearer API_KEY"

# 4. Check ASSURE-CODE received event
# 5. Verify specs regenerated
# 6. Check GitHub PR created
```

### Step 4: Enable Auto-Send (5 min)
```bash
# In scraper .env
ASSURE_CODE_AUTO_SEND=true

# Now every completed job automatically
# sends to ASSURE-CODE
```

### Step 5: Monitor Integration (5 min)
```sql
-- Check sent events
SELECT * FROM audit_logs 
WHERE action = 'sent_to_assure_code'
ORDER BY timestamp DESC;

-- Check received updates
SELECT * FROM audit_logs 
WHERE action = 'specification_updated'
ORDER BY timestamp DESC;
```
